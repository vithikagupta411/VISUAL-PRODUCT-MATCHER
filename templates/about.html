About Visual Product Matcher

The Visual Product Matcher is an AI-powered web application that finds visually similar products based on an uploaded image or an image URL.

🔹 It uses deep learning embeddings from a pretrained model to compare product images.
🔹 The system stores product details in a database and performs similarity searches in real time.
🔹 Built with Flask, PyTorch, and OpenAI CLIP.

This project demonstrates how computer vision can help e-commerce platforms provide better product discovery and recommendations.
